About

This countdown timer shows the time remaining until October 1st, 2025 in Amsterdam timezone.

The timer updates every minute and displays days, hours, and minutes remaining.

API Endpoints:

  • /api/countdown.png - Static PNG image
  • /api/countdown.gif - Animated GIF
  • Query parameters: ?width=600, ?dpr=2
Image refreshes every 60 seconds
